  1. 在浏览器中安装 DeepSeek AI 扩展
  2. 点击工具栏中的扩展图标
  3. 输入您的 DeepSeek API 密钥
  4. 选择您偏好的回答语言
  5. 在任意网页上选择文本，开始与 AI 对话！

使用方法

文本选择使用

在网页上选中任意文本后，点击旁边的 DeepSeek 图标，即可呼出对话窗口，获取智能回答。

快捷键使用

无论是否选中文本，都可以直接使用快捷键 Ctrl+Shift+D（Windows）或 Command+Shift+D（Mac）呼出对话窗口。

功能特点

智能对话

• 支持多轮对话，记住上下文

• 实时流式响应，打字机效果

• 支持重新生成回答

界面交互

• 可拖拽调整窗口位置和大小

• 支持 Markdown 格式化显示

• 支持 LaTeX 数学公式渲染

• 代码块一键复制

个性化设置

• 自定义语言偏好

• 深色模式自动适配

• 可配置快捷键

使用技巧

💡 使用快捷键可以更快地打开对话窗口，Mac 为 Command + Shift + D，Windows/Linux 为 Ctrl + Shift + D。

💡 点击代码块右上角的复制按钮，可以快速复制代码片段

💡 如果对 AI 的回答不满意，可以点击重新生成按钮获取新的答案

隐私说明

我们重视您的隐私。DeepSeek AI 扩展只会在必要时发送您选中的文本到 API，不会收集或存储任何其他个人信息。您的 API 密钥仅保存在本地浏览器中。
